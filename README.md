# 🧠 How It Works

The system works in three main steps.

### 1️⃣ User Interaction (WhatsApp)

Customers interact with the bot using commands such as:

```
.buy
```

The bot then shows available **Bingwa Sokoni data bundle packages**, for example:

- 250MB
- 500MB
- 1GB
- 2GB

The user selects the desired bundle.

---

### 2️⃣ Payment via M-Pesa

After selecting a bundle:

1. The user enters their phone number.
2. The bot sends an **M-Pesa STK Push** request.
3. The user confirms the payment using their **M-Pesa PIN**.

Once the payment is completed:

- M-Pesa sends a **payment callback** to the server.

---

### 3️⃣ Automatic Bundle Delivery

After successful payment:

A background **automation app** listens for the **M-Pesa confirmation message**.

These automation apps can:

• Detect the payment SMS  
• Trigger a **USSD command**  
• Automatically send the **Bingwa Sokoni bundle**

Examples of automation apps include:

- USSD automation apps
- Android automation tools
- Task based automation systems

Each vendor can choose the automation tool that works best for them.
